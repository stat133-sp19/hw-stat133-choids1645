Binomial

The package ‘“binomial”’ is a implementation for calculating probabilities, creating a data frames, and visualizing the probabilities as the number of successes increase.

bin_Choose(n, k)

This calculates “n choose k”.

c <- bin_choose(10, 5)
c
#> [1] 252

bin_probability(success, trials, prob)

This function calculates the probabilities of getting given number of successes out of given trials where probability of one success is given as well.

p <- bin_probability(2, 5, 0.5)
p
#> [1] 0.3125

##bin_distribution(trials, prob) This creates a data frame of the number of successes(0 to trials) and corresponding probabilities. The returned data frame is a ‘“bindis”’ object. Using plot() we can draw a plot from the bindis object

bd <- bin_distribution(5, 0.5)
bd
#>   success probability
#> 1       0     0.03125
#> 2       1     0.15625
#> 3       2     0.31250
#> 4       3     0.31250
#> 5       4     0.15625
#> 6       5     0.03125
plot(bd)

##bin_cumulative(trials, prob) This creates a data frame of the number of successes(0 to trials), corresponding probabilities, and cumulative probabilities. The returned data frame is a ‘“bincum”’ object. Using plot() we can draw a plot from the bincum object

bc <- bin_cumulative(5, 0.5)
bc
#>   success probability cumulative
#> 1       0     0.03125    0.03125
#> 2       1     0.15625    0.18750
#> 3       2     0.31250    0.50000
#> 4       3     0.31250    0.81250
#> 5       4     0.15625    0.96875
#> 6       5     0.03125    1.00000
plot(bc)

##bin_variable(trials, prob) This creates a “‘binvar’” object with number of trials and probability of getting a success. Using summary() method, we can print out summary of binomial variable object

bv <- bin_variable(5, 0.5)
bv
#> "Binomial variable"
#> 
#> Parameters
#> - number of trials: 5
#> - prob of success : 0.5
summary(bv)
#> "Summary Binomial"
#> 
#> Parameters
#> - number of trials: 5 
#> - prob of success : 0.5 
#> 
#> Measures
#> - mean    : 2.5 
#> - variance: 1.25 
#> - mode    : 3 2 
#> - skewness: 0 
#> - kurtosis: -0.4

##bin_mean(trials, prob) This returns the expected value

bin_mean(5, 0.5)
#> [1] 2.5

##bin_variance(trials, prob) This returns the variance

bin_variance(5, 0.5)
#> [1] 1.25

##bin_mode(trials, prob) This returns the mode

bin_mode(5, 0.5)
#> [1] 3 2

##bin_skewness(trials, prob) This returns the skewness

bin_skewness(5, 0.5)
#> [1] 0

##bin_kurtosis(trials, prob) This returns the kurtosis

bin_kurtosis(5, 0.5)
#> [1] -0.4
